## 开发运行

- 容器运行
  ```bash
  docker run --rm -it --name Node_Code \
  --cpus=1 --memory=1024m --network host \
  -w /app -v .:/app node:6.17.1-alpine ash
  ```
- NPM 换源
  ```bash
  npm config set registry https://registry.npmmirror.com
  ```
- 安装依赖
  ```bash
  npm i
  ```
- 运行
  ```bash
  PORT=1234 npm start
  ```
